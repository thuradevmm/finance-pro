# This is the project features detail that you should know before developing the source code.

### Project Scope Document
### Personal Financial Management System
## Project Objective
The objective of this project is to develop a web-based personal financial management system that helps the client manage and monitor personal financial data in one place.
The system will replace the client’s current Excel-based tracking process and reduce manual calculations by providing structured data entry, automatic balance calculation, budgeting, debt tracking, savings tracking, subscription tracking, reports, and dashboard summaries.
The system will be designed as a modern, professional, and minimalist web application that can be accessed through a browser on desktop, tablet, and mobile screen sizes.
________________________________________
## User Roles
2.1 Owner / Main User
The owner is the main user of the system. The owner can manage personal financial records, view reports, update settings, and export data.
The owner can:
- Log in and manage personal profile
- Add, edit, and delete financial records
- Manage accounts and wallets
- Manage income and expense categories
- Manage budgets
- Manage debts and repayments
- Manage savings goals
- Manage subscriptions
- Upload receipts and related files
- View dashboards and reports
- Export financial data
- Manage transactions
- Manage categories
- Manage accounts and wallets
- Manage budgets
- Manage uploaded files
- Manage reports
- Manage system settings
